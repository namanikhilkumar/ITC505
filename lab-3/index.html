Steps to Deliver a Custom Font:

To deliver a specific font to clients, i have followed the steps discussed below.

then i have chosen a font of my desired choice from a a website called dafont, then i have downloaded a font called "Game Of Squids" in a zip format and unzip the file,

after copy the copy the "Game Of Squids.otf" file in our source folder of our to make it access.

Then access index.css file and we can use the @font-face rule in your CSS to define the font and specify the font file’s location.

To deliver a specific font to clients you can then apply the font to specific HTML elements using the font-family properly.

And I also configured the source URL by using the following code src: URL(“Game Of Squids.off”) format(“woff”);

we are formatting “.otf” file to “woff2” because every major browser supports this version.
